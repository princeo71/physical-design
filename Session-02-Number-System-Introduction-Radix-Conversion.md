# Introduction to Radix Conversion

- Apply radix conversion techniques to practical problems.

## Radix Conversion Formats
To convert numbers from one base to another, you can follow the step-by-step processes outlined below:

### Decimal to Binary Conversion
1. **Divide the decimal number by 2.** Write down the quotient and the remainder.
2. **Repeat** the process for the quotient until it reaches zero.
3. The binary equivalent will be the remainders read in reverse order.

### Binary to Decimal Conversion
1. **Multiply each bit** by 2 raised to the power of its position (starting from 0).
2. **Sum up** all the results to get the decimal equivalent.

### Binary to Hexadecimal Conversion
1. **Group the binary number** into sets of four (starting from the right)
2. **Convert each group** into its hexadecimal equivalent.

### Common Mistakes
- Miscounting bits when converting from binary to decimal.
- Forgetting to carry over when performing decimal to binary conversions.
